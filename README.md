プロ生LTタイマー
==============

IT勉強会などで利用するためのLTタイマーです。
プロ生ちゃんが音声付きでカウントダウンしてくれます。

このアプリは以下のガイドラインに沿い、プロ生ちゃんの画像及び音声を二次利用しています。
